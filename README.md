# McDonald's Review Analysis
An R-based text mining and NLP project that analyzes McDonald's customer reviews using TF-IDF scoring, word clouds, LDA topic modelling, and sentiment analysis.

- Code: Aleksander Mroz 
- Documentation: Piotr Podobinski

### Required Libraries
Install all dependencies by running this in your R console:
 
```r
install.packages(c(
  "tm",
  "wordcloud",
  "topicmodels",
  "tidyverse",
  "tidytext",
  "textstem",
  "ggplot2",
  "ggthemes"
))
```
